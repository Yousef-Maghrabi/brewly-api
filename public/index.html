About the API

The Brewly API provides endpoints for fetching coffee details, machine specs, and more. Whether you’re building a store, dashboard, or recommendation system, Brewly helps you serve data that fuels the coffee world.

Our RESTful API is easy to integrate, JSON-based, and secured with token authentication.
